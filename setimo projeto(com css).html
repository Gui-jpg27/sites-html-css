CONHECENDO PROPRIEDADES

PELO QEU EU VI,PARECE QUE QUANDO SE VAI MUNDAR ALGUMA COISA NO TEXTO,COLOCA FONTE

ENTAO EU IREI COLOCAR AQUI AS PALAVRAS QUE SAO SEGUINTES A FONTES:

FONT-WEHDTH:ESPEFICICA A GROSURA DAS PALAVRAS DO TEXTO

FONT-SIZE:ALTERA O TAMANHO DA FONTE

FONT-FAMILY:MUDA O ELEMENTO DO TEXTO,QUER DIZER,MUNDA A FORMA DAS PALAVRAS

FONT-STYLE:ELE É QUASE PARECIDO COM O FAMYLI,MAS ELE REFLETE MAIS AQUELES QUE APARECE NO WORD.

TEXT DECORATION:APRESENTA UM UNDERLINE NA PALVRA,AÍ ELE VAI TER QUATRO SEGUIMENTOS:EM CIMA(OVERLINE),NO CENTRO(LINE-THROUGH),EM BAIXO(UNDERLINE),SEM NENHUM TRAÇO(NONE)

TEXT-ALING: SERVE PARA ALINHAR O TEXTO,E ELE POSSUI TRES SEGUIMENTOS: CENTER(CENTRO),LEFT(PARA A ESQUERDA),RIGTH(PARA A DIREITA)

LINE-HEIGTH:SERVE PARA AUMENTAR OU DIMINUIR A EXPESSURA DAS LINHAS DO TEXTO.
